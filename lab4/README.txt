Последовательность действий:
	1) 
	Командой
		make
	комплируется выходной файл main.out
	
	2) 
	Командой
		main.out Nh Nv L
	где Nh, Nv = 1, 2, 4, 8 ... (по умолчанию 2) - стороны "ячейки",
	L - количество нейронов
	---
	После вызова запускается программа, считываются изображения из img_in/,
	сеть обучается по первой картинке, далее через нее прогоняются все картинки.
	Выходные результаты записываются в img_out/ 
	Выходы, соответствующие шагам обучения, записываются в fit_steps/
